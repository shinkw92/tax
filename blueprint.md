
# Lotto Number Generator

## Overview

This project is a simple web application that generates and displays random lottery numbers. The user can click a button to generate a new set of numbers.

## Features

-   Generate 6 unique random numbers between 1 and 45.
-   Display the generated numbers in a visually appealing way.
-   A button to generate a new set of numbers.

## Design

-   **Layout:** A centered container with the title, the generated numbers, and the button.
-   **Typography:** Clear and readable font for the numbers.
-   **Color Scheme:** A simple and clean color scheme.
